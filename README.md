# 🇧🇷 Amazon Brazil — Last-Mile Delivery Analytics

## 📌 Project Overview
This repository contains a **data analytics case study** focused on analyzing
last-mile delivery performance for Amazon Brazil using **SQL, datasets, and
business-focused insights**.

The project evaluates delivery delays, partner performance, and operational
efficiency to identify improvement opportunities.

## 📊 What’s Included
- Raw and processed datasets
- SQL queries for KPI calculation and analysis
- Insight summary derived from analysis
- Final presentation for stakeholder communication

## 🛠 Tools Used
- SQL
- Excel / CSV datasets
- PowerPoint / PDF for presentation

## 📁 Repository Structure
- `data/` → datasets used for analysis
- `sql/` → analysis queries
- `analysis/` → insights & findings
- `presentation/` → final presentation
